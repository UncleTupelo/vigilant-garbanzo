This struct is passed as an argument to create a new instance of an [Overseer].

As any entity that satisfies the interface may act as a Subsystem this allows mocking in the test code:

Each Subsystem is supposed to implement some interface that is generic over message type that is specific to this Subsystem. At the moment not all subsystems are implemented and the rest can be mocked with the [DummySubsystem].
